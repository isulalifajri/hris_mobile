# hris_mobile

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter application.

create project : `flutter create hris_mobile`

cara menjalankan emulator nya gini:
```
1️. Buka Android Studio
2️. More Actions → Device Manager
3️. Klik `Start` di salah satu emulator
4️. Tunggu sampai emulator nyala full
```
cek : `flutter devices`
jika muncul : `Android SDK built for x86 (mobile)`

lalu jalankan: `flutter run` or `flutter run -d emulator-5554`
